# Clinical-Administration
The Clinical Administration project was my first project. Which I developed during my BCA. The entire project has been developed during Java, and SQL has been utilized as the database.

In this project features various clinic management capabilities, including adding, discharging, and editing patient records; 
adding, deleting, and editing doctor and staff profiles; as well as booking, rescheduling, viewing, and editing appointments, among other features.
